# Intent Labels

Danh sách intent ban đầu của MeowAI.

| Intent | Ý nghĩa | Ví dụ |
| --- | --- | --- |
| `hoi_gia` | Khách hỏi giá, bảng giá, gói rẻ | "gói nào rẻ nhất" |
| `hoi_ton_kho` | Khách hỏi còn hàng hay không | "còn hàng không" |
| `tu_van_san_pham` | Khách cần gợi ý sản phẩm theo nhu cầu | "em cần gói chạy web nhỏ" |
| `thanh_toan` | Khách hỏi cách thanh toán | "ck được không" |
| `giao_hang` | Khách hỏi giao hàng/bàn giao | "bao lâu nhận được" |
| `doi_tra` | Khách hỏi đổi trả/hoàn tiền | "mua rồi đổi được không" |
| `bao_hanh` | Khách hỏi bảo hành | "bảo hành bao lâu" |
| `khuyen_mai` | Khách hỏi coupon/khuyến mãi | "có mã giảm giá không" |
| `thong_tin_shop` | Khách hỏi thông tin shop | "shop ở đâu" |
| `fallback` | Không rõ ý định | "abc gì đó" |
